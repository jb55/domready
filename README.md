domReady component
--------

It's easy. Works like this:
``` js
var domready = require('domready') // would be jb55-domready from root
domready(function () {
  // dom is loaded!
})
```

Browser support
---------------

  * IE6+
  * Firefox 2+
  * Safari 3+
  * Chrome *
  * Opera *

Installing
--------

    $ component install jb55/domready
